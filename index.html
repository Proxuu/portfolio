Cześć! Nazywam się Kacper Zdziebło i jestem pasjonatem programowania oraz technologii webowych. Od kilku lat zgłębiam tajniki tworzenia stron i aplikacji internetowych, łącząc kreatywność z techniczną precyzją. Moją największą motywacją jest tworzenie projektów, które są zarówno funkcjonalne, jak i estetyczne. Specjalizuję się w budowie responsywnych stron internetowych, aplikacji oraz projektach backendowych. Uwielbiam pracować nad nowymi wyzwaniami, rozwijać swoje umiejętności i wdrażać innowacyjne rozwiązania.

Moje umiejętności:

Frontend Development

Zaawansowana znajomość HTML5, CSS3 (w tym CSS Grid i Flexbox) oraz preprocesorów takich jak SASS/SCSS.

Tworzenie dynamicznych interfejsów użytkownika z wykorzystaniem JavaScript (ES6+) i React (Hooks, Redux, Context API).

Doświadczenie z TypeScript w projektach frontendowych.

Umiejętność budowania responsywnych i dostępnych stron internetowych (RWD, WCAG).

Backend Development

Tworzenie i zarządzanie API REST i GraphQL z użyciem Node.js i Express.

Praca z bazami danych: relacyjnymi (MySQL, PostgreSQL) oraz nierelacyjnymi (MongoDB, Redis).

Implementacja uwierzytelniania i autoryzacji (OAuth2, JWT, Passport.js).

Doświadczenie w pracy z systemami kolejkowymi (np. RabbitMQ, Kafka).

DevOps i narzędzia

Praca z systemami kontroli wersji (Git, GitHub, GitLab).

Automatyzacja CI/CD (GitHub Actions, Jenkins, Docker).

Podstawowa znajomość konteneryzacji (Docker, Docker Compose).

Wdrażanie aplikacji w środowiskach chmurowych (AWS, Heroku, Vercel).

Inne umiejętności

Znajomość testowania jednostkowego i integracyjnego (Jest, Mocha, Cypress).

Praktyka w projektowaniu architektury systemów i optymalizacji kodu.

Umiejętność pracy w metodykach Agile (Scrum, Kanban).

Moje główne projekty

ForestLife

ForestLife to przeglądarkowa gra survivalowa multiplayer, w której gracze zostają wrzuceni do dzikiego lasu pełnego niebezpieczeństw. Ich głównym celem jest przetrwanie – muszą zbierać zasoby, budować schronienia, tworzyć narzędzia i bronić się przed atakami innych graczy. W grze można eksplorować, walczyć, a także współpracować z innymi, tworząc sojusze.

GitHub

ezMaturka

ezMaturka to interaktywna platforma edukacyjna dla maturzystów, oferująca zadania z arkuszy maturalnych z matematyki na poziomie podstawowym. Uczniowie mogą rozwiązywać zadania, korzystać z podpowiedzi i pełnych rozwiązań, a także zapisywać swoje obliczenia na wirtualnej tablicy. Platforma pomaga w efektywnej nauce i przygotowaniu do egzaminu maturalnego.

GitHub

Sówka

Sówka to nowoczesna platforma edukacyjna, która umożliwia przeglądanie podręczników szkolnych z różnych przedmiotów i poziomów nauczania. Strona zapewnia wygodny dostęp do wszystkich potrzebnych podręczników w jednym miejscu, eliminując konieczność posiadania ich w fizycznej formie.
